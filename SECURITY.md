# Security Policy

## Authorized use only

This tool is intended **exclusively for authorized security testing** —
red-team and resilience engagements where you have written authorization from
the site operator and CaptchaFox (Scoria Labs GmbH) to test the target.

## Reporting findings

If your testing reveals a bypass, coordinate disclosure with the vendor before
any public discussion. Open a **private** GitHub Security Advisory for issues
in this library itself.
